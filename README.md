# board_game_with_socket_MFC

보드게임을 구현했다. 소켓 통신을 이용하여 클라이언트 측에서 서버 측으로 주사위 값과 통행료를 계산하고 각 서버에 저장시킨 클라이언트들의 부동산을 바탕으로 통행료를 받는 클라이언트와 통행료를 내는 클라이언트들의 이동을 send와 receive 함수를 통해 parsing을 하여 값을 전달했다. 예를 들어 주사위 값은 크기가 2~12로 (char)2자임에서 착안하여, 클라이언트가 접속하는 말의 데이터는 (char) 3자로 했고(ex> data : 100 -> 1번 말, data : 200 : -> 2번 말...etc...), 금액은 최소한 (char)4자이기 때문이다. 이 것으로 보드게임을 만들었다.
